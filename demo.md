# easymd 演示文档

这是一个 **演示文件**，用来试用 easymd 的编辑和预览。

## 功能列表

- 左侧 vim 式编辑
- 右侧实时 Markdown 预览
- 光标移动时预览联动滚动

## 代码块

```python
def hello(name: str) -> str:
    return f"Hello, {name}!"
```

## 表格

| 按键 | 作用 |
| ---- | ---- |
| `i`  | 进入插入模式 |
| `dd` | 删除当前行 |
| `:w` | 保存 |

> 引用块也可以正常渲染。

试着按 `i` 开始编辑，右边的预览会自动刷新。
