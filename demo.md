# easymd 功能演示与练习

这份文档既是 **Markdown 渲染** 的展示，也是一份 **vim 操作练习册**。
左侧编辑、右侧预览，移动光标时预览会联动滚动。下面每一节都标注了可以
在该处试用的按键，照着练一遍就能覆盖 easymd 的大部分功能。

> 提示：先按 `i` 进入插入模式随意改几个字，按 `Esc` 回到普通模式，
> 右侧预览会在停手约 0.25 秒后自动刷新。

---

## 1. 移动与查找

下面这行逗号很多，适合练 `f` / `t` 和 `;` `,`：

    apple, banana, cherry, durian, elderberry, fig, grape

- `f,` 跳到下一个逗号，按 `;` 继续跳，`,` 反向跳
- `t,` 停在逗号前一格
- `df,` 删到逗号（含），`dt,` 删到逗号前
- `3f,` 直接跳到第三个逗号

整段文字里 `preview` 这个词出现了多次：preview 渲染、preview 滚动、
preview 刷新。把光标放到任意一个 `preview` 上，按 `*` 向后搜索、`#` 向前
搜索，再用 `n` / `N` 在多个 preview 之间跳转。

## 2. 括号配对（`%`）

把光标放在任意括号上按 `%` 会跳到配对的另一半，支持嵌套：

```python
def fib(n):
    if n < 2:
        return n
    return fib(n - 1) + fib(n - 2)


data = {"users": [{"id": 1}, {"id": 2}], "count": (2, 0)}
```

- 光标在 `def fib(` 的 `(` 上按 `%` → 跳到对应 `)`
- 在 `{"users"` 的 `{` 上按 `%` → 跳到整个字典结尾的 `}`
- `d%` 可以连同配对括号一起删除

## 3. 段落移动（`{` `}`）

下面是几段被空行分隔的文字，用 `}` 向下、`{` 向上在段落间跳：

第一段。这里随便写点内容，让段落有一定长度，方便观察光标跳动的效果。

第二段。`}` 会停在下一个空行，`d}` 删到段落末尾，`2}` 一次跨两段。

第三段。段落移动配合 operator 很常用，比如 `y}` 复制一整段。

## 4. 编辑与重复（`.`）

| 操作 | 按键 | 说明 |
| --- | --- | --- |
| 删除字符 | `x` | `3x` 删三个；`.` 重复 |
| 替换字符 | `r` | `rX` 把当前字符换成 X |
| 改写单词 | `cw` | 改完按 `Esc`，移到下个词按 `.` 重复 |
| 改写内部 | `ciw` `ci"` `ci(` | 文本对象，改引号/括号内的内容 |
| 整行操作 | `dd` `yy` `cc` | 删除 / 复制 / 改写整行 |
| 行尾操作 | `D` `C` `Y` | 删到行尾 / 改到行尾 / 复制行 |
| 合并行 | `J` | 把下一行接到当前行尾 |
| 粘贴 | `p` `P` | 在后 / 前粘贴 |

练习 `.`：把光标放到下面任意一行的 `TODO` 上，按 `cwDONE` 然后 `Esc`，
再移动到下一个 `TODO` 按 `.` 即可重复同样的修改。

## 7. 保存与退出

- `:w` 保存，`:w 新文件名` 另存
- `:q` 退出（有未保存修改会拒绝），`:q!` 强制退出
- `:wq` 或 `:x` 保存并退出
- `/文字` 搜索后 `n` / `N` 跳转

改完记得用 `:w` 保存，或 `:q!` 放弃所有修改退出。
